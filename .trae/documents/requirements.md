# 需求文档

## 1. 项目概述

项目名称：工单管理与图表展示页面（简化版）

## 2. 功能需求

### 2.1. 登录页面
- **表单字段**: 用户名、密码。
- **登录逻辑**:
  - 用户名为 "admin" 时，视为管理员。
  - 其他用户名，视为普通用户。
- **页面跳转**: 登录后跳转到主页面，无需实现 Token 存储。

### 2.2. 工单列表
- **数据展示**: 使用表格组件展示工单数据。
- **表格字段**:
  - `id`: 工单ID
  - `project`: 项目名称
  - `overtime`: 是否加班
  - `hours`: 工时
  - `created_at`: 创建时间
- **初始数据**: 初始化4-5条模拟数据。
- **删除功能**:
  - 每行提供一个“删除”按钮。
  - “删除”按钮仅管理员可见。
  - 删除记录后，图表需要自动更新。

### 2.3. 图表展示
- **图表类型**: 柱状图。
- **图表标题**: "Project Hours Distribution"。
- **图表内容**: 按项目名称分组，统计每个项目的累计工时。

### 2.4. 权限控制
- **管理员 (admin)**:
  - 可以看到并使用“删除”按钮，删除工单记录。
- **普通用户**:
  - 无法看到“删除”按钮。

## 3. 技术栈

- **框架**: Vue 3
- **UI 组件库**: Ant Design Vue
- **图表库**: ECharts
- **样式**: Less 或 Tailwind CSS

## 4. 交付要求

- 项目代码。
- 一个可运行的在线预览地址。
- 一份说明文档，包含 AI 工具使用情况等。
